## Current Version
0.2.0

## Highlights
- Timer now survives a refresh — running, paused, or alarm-pending state is fully restored.
- Negative overflow clock counts up during alarm so you can see how long you've been ignoring it.
- Notification prompt now appears on first Start with a plain explanation before asking.
- Snooze countdown shown in status text instead of a click instruction.
- Phase labels throughout: Working / Breaking while running; next phase on alarm and skip buttons.
- Stars no longer chase a stale cursor after switching windows.
